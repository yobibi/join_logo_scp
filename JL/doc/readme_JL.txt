-----------------------------------------
JLフォルダ構成
-----------------------------------------
join_logo_scp ver4.0でサブフォルダからの読み込みに対応。

従来ファイルそのままでも動作するが、
サブフォルダを利用して下記フォルダ構成にしている。
（.は起動時に指定したフォルダ、それ以外はその場所からのフォルダ）

.      : 起動時に指定できるファイル
common : Callで呼び出されるファイル
           JL_ini_*  : 起動用の設定を記載
           JL_out_*  : 追加出力するファイルの設定
           JL_set_*  : 設定して元のファイルに戻る
           JL_sub_*  : 関数定義を記載
user   : ユーザー作成置き場(ver5.0追加)
           JL_run_*  : 全体記載。選択スクリプトから呼び出しを想定
           他フォルダのスクリプトファイルを置いたらここが読み込み優先される
doc    : 説明文
data   : Read系コマンドで使用する定型のデータリスト(ver4.1追加)
