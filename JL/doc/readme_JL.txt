-----------------------------------------
JLフォルダ構成
-----------------------------------------
join_logo_scp ver4.0でサブフォルダからの読み込みに対応。

従来ファイルそのままでも動作するが、
サブフォルダを利用して下記フォルダ構成にしている。
（.は起動時に指定したフォルダ、それ以外はその場所からのフォルダ）

.      : 起動時に指定できるファイル
common : Callで呼び出されるファイル
           JL_out_*  : 追加出力するファイルの設定
           JL_set_*  : 設定して元のファイルに戻る
doc    : 説明文
